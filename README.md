# Projeto 1 - Binary to Decimal / Decimal to Binary

## Objetivo
Fazer um sistema o qual irá retornar um número Binário ou Decimal dependendo
da Solicitação.
Seria somente em 1 bloco de solicitação e poderia colocar a solicitação 
em **qualquer** *input*.
